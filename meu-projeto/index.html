O que é uma Expressão Lambda?

Uma função lambda é uma forma compacta de escrever funções anônimas em Java. Ela permite passar um bloco de código como argumento para uma função ou método, sem a necessidade de criar uma classe anônima separada. Antes do Java 8, era comum usar classes anônimas para passar comportamentos como parâmetros, o que resultava em código verboso. Com as lambdas, o código se torna mais simples e direto.

Sintaxe Básica

A sintaxe de uma função lambda é composta por três partes principais:

  • Parâmetros: Definem o que a função vai receber.
  • Seta (->): Indica a separação entre os parâmetros e o corpo da função.
  • Corpo: Define o que a função vai executar.
